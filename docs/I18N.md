# Architecture multilingue IronMarkGear — V77

Langues prévues : français, néerlandais, anglais.

## Fonctionnement actuel

- Sélecteur visible dans l’en-tête : FR / NL / EN.
- La langue est sauvegardée dans le cookie `img_lang` et dans `localStorage`.
- La langue choisie reste active sur les pages suivantes et après réouverture du site dans le même navigateur.
- Les textes Next.js principaux utilisent le dictionnaire `lib/i18n.ts`.
- Les produits existants ont une traduction FR/NL/EN côté code.
- Les produits ajoutés via l’admin utilisent le français comme fallback tant que la traduction automatique/API n’est pas branchée.

## Image de la page location

Le visuel location reste une image. Pour rendre la traduction possible sans régénérer l’image à chaque langue, la V77 ajoute un bloc texte HTML superposé et traduisible.

Plus tard, il sera possible de brancher une variante d’image par langue si le texte intégré dans l’image doit aussi changer.

## Prochaine étape API

Quand l’hébergement + backend seront prêts, une route API pourra générer automatiquement les traductions depuis le français vers NL/EN.
